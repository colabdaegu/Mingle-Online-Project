# 둥글게 둥글게 온라인
![Preview](.git-Images/Mingle_icon.ico)  
> 음악이 멈추는 순간, 생존을 건 사투가 시작된다  
> 제한된 시간, 정해진 인원… 정확히 맞춘 방만 생존!

---

## 🕹️ 게임 소개
이 게임은 **오징어 게임2 ‘짝짓기 게임’ 콘셉트를 기반으로 제작된 생존형 배틀로얄** 게임입니다.  
플레이어는 음악이 흐르는 동안 중앙 홀에서 대기하다가, 음악이 멈추는 순간 무작위로 제시되는 인원 수에 맞춰 방으로 이동해야 합니다.  
시간 내에 방에 들어가지 못하거나, 방 인원이 조건에 맞지 않으면 탈락하게 됩니다.  

본 게임은 **인공지능 모드와 온라인 플레이**를 지원합니다.
